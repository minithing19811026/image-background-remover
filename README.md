# Image Background Remover

A modern web application to remove backgrounds from images instantly using AI.

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **API**: remove.bg via Cloudflare Workers

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Cloudflare Pages

1. Connect your GitHub repository to Cloudflare Pages
2. Build settings:
   - Build command: `npm run build`
   - Build output directory: `.next`
